genericpath module¶

Path operations common to more than one OS Do not use directly. The OS specific modules import the appropriate functions from this module themselves.

genericpath.commonprefix(m)[source]¶

Given a list of pathnames, returns the longest common leading component

genericpath.exists(path)[source]¶

Test whether a path exists. Returns False for broken symbolic links

genericpath.getatime(filename)[source]¶

Return the last access time of a file, reported by os.stat().

genericpath.getctime(filename)[source]¶

Return the metadata change time of a file, reported by os.stat().

genericpath.getmtime(filename)[source]¶

Return the last modification time of a file, reported by os.stat().

genericpath.getsize(filename)[source]¶

Return the size of a file, reported by os.stat().

genericpath.isdir(s)[source]¶

Return true if the pathname refers to an existing directory.

genericpath.isfile(path)[source]¶

Test whether a path is a regular file

genericpath.samefile(f1, f2)[source]¶

Test whether two pathnames reference the same actual file

genericpath.sameopenfile(fp1, fp2)[source]¶

Test whether two open file objects reference the same file

genericpath.samestat(s1, s2)[source]¶

Test whether two stat buffers reference the same file
