Overview

We have a central object for creating and retrieving loggers. This retains a local reference to all loggers created. This is so an existing logger can be looked up by name, but also to prevent loggers from being garbage collected. A dRuby reference to an object is not sufficient to prevent it being garbage collected!

def get_logger(name)
    if !@loggers.has_key? name
        # make the filename safe, then declare it to be so
        fname = name.gsub(/[.\/\\:]/, "_").untaint
        @loggers[name] = Logger.new(name, @basedir + "/" + fname)
    end
    return @loggers[name]
end
